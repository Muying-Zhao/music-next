# 官方推荐

node-sass 基于的 libsass 已经不再维护，并且 sass（Dart Sass）是用纯 JavaScript 编写的，更易于安装和维护。

每次切换注意刷新一遍，触摸需要重新刷新使用

## Project setup

```
npm install
npm i -D sass
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
